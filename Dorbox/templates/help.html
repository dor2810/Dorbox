How Does Dorbox Work?


Dorbox is divided into two parts
  • Groups
  • Private Chats

Groups

Creating Groups

You can create a group by pressing the Create Group button.
Inside these group, members with permission will be able to upload files and create folders.
When you create a group you are set to be the Group Admin.

Each group has 3 levels of permissions
  • Reader
  • Editor
  • Admin

Reader

Reader is the most basic permission
A reader can only read the descriptions of files and folders and download files
You can leave a group whenever you want

Editor

Editor is above Reader and can do anything reader does and more.
Editor additional options:
  • Can Create new folders and upload new files
  • Can edit all the files and the folders
  • Can only delete files that he uploaded or folders that he created

Admin

Admin is above Reader and Editor and can do anything reader and editor do and more.
Admin additional options:
  • The only member who can invite new members or remove members from the group
  • The only member who can change others permission in the group
  • Can edit and delete each file and folder within the group

Private Chats

Private Chats are the place to share files between only two people
You can create a new Private Chat by creating with a username or by clicking on Private Chat button inside the Members And Permissions Page
Inside of a private chat each member has all the options from uploading new file to deleting it.
